Installation Guide

Method 1 — Manual

  1. Download the ZIP file.
  2. Extract it anywhere on your PC.
  3. Open Chrome → Extensions.
  4. Enable Developer Mode.
  5. Click "Load Unpacked".
  6. Select the extracted folder.
