Installation Guide

  1. Download the ZIP file.
  2. Extract it anywhere on your PC.
  3. Open Chrome → Extensions → Enable Developer Mode.
  4. Click "Load Unpacked" and select the extracted folder.
  5. Done. Open a new tab.
