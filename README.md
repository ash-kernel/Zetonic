**Zetonic** is a Chrome extension that replaces your default new tab with a clean, distraction-free experience featuring anime-themed backgrounds and bold time display.

---

## Features

- **Video & Image Backgrounds** - Curated anime videos, random images, or your own custom URL
- **Bold Time Display** - Clean typography with seconds precision
- **24h/12h Format** - Toggle between formats
- **Personalized Greeting** - "Good morning, [name]" - set your name
- **Daily Focus** - One-word intention above the clock
- **Quick Links** - Customizable bookmarks
- **Search** - Google or DuckDuckGo integration
- **Quotes** - Random motivational quotes
- **Weather** - Location-based weather display
- **Themes** - Default, Amber, Cyan, Rose, Violet
- **Focus & Zen Modes** - Minimalist experiences

## Install

**Option A: Run the installer**
- Download `Zetonic-Installer.exe` from Releases
- Run the exe and follow on-screen instructions

**Option B: Load unpacked**
1. Download this repository
2. Navigate to `chrome://extensions/`
3. Enable **Developer Mode**
4. Click **Load unpacked**
5. Select the Zetonic folder
6. Click extension icon to configure settings

## Settings

All settings are accessible via the extension popup:
- Personal: name, daily focus
- Display: clock, quote, weather, quick links, focus/zen mode, time format, theme
- Background: video/image mode, custom URL, rotation, blur
- Video management: add URLs, upload local videos
- Search: toggle, engine selection
- Quick links: add/edit bookmarks

## Support

- Google Chrome
- Microsoft Edge
- Brave Browser
- Any Chromium-based browser
